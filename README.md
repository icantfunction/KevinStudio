# Worldwide Studio Space

Production-ready website for a studio and event venue rental business. Built with Next.js App Router, TypeScript, and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Create a `.env.local` file and set:

```
HONEYBOOK_BOOKING_URL="https://booking.honeybook.com/your-link"
HONEYBOOK_INQUIRY_URL="https://forms.honeybook.com/your-inquiry"
```

## Structure

- `src/app`: Routes and pages
- `src/data`: Content models (gallery, rates, amenities, testimonials)
- `src/components`: Layout components (header, footer, sticky CTA)
- `public/media`: Placeholder imagery
